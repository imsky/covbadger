# covbadger

![Coverage](https://rawgit.com/imsky/covbadger/master/coverage.svg)

`covbadger` generates code coverage badges.

## Usage

```
# generate a badge that shows 97% coverage
covbadger 97

# pipe to covbadger
echo 97 | covbadger -
```
